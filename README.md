### Description
This Laravel project implements a BookStock library management system. It includes:

- **Index and Create operations** for `books`, `categories`, and `authors`.
- **Database migrations** for `books`, `categories`, and `authors`.
- **Seeders** with fake data and example images.
- **File uploads** for book covers stored in `storage/app/public/covers`.
